Entidades y Atributos

Cliente: ID_Cliente (Clave), Nombre, Teléfono, Correo, Puntos_Acumulados.
Sala: ID_Sala (Clave), Número_Sala, Capacidad_Butacas.
Película: ID_Pelicula (Clave), Título, Duración, Clasificación.
Función: ID_Funcion (Clave), Fecha, Hora, Entradas_Vendidas.
Venta (Entrada): ID_Venta (Clave), Fecha_Venta, Precio_Final, Es_Gratuita.

Relaciones

Una sala tiene muchas funciones (1:N). Una función pertene a una sola Sala.
Una película se proyecta en muchas funciones (1:N). Una función tiene una sola película.
Un cliente puede realizar muchas ventas (1:N). Una venta pertenece a un solo cliente.
Una función puede tener muchas ventas (1:N). Una Venta corresponde a una única función.
